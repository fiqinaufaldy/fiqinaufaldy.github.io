About WebSite

Informassi tentang website ini

Ini adalah website portofolio yang saya buat untuk menampilkan semua proyek-proyek dan keterampilan yang saya miliki. Tujuannya adalah untuk mewujudkan semua kemampuan yang saya kembangkan selama perjalanan pembelajaran saya dalam pemrograman. Saat ini, website ini dibangun menggunakan HTML, TailwindCSS, dan JavaScript. Meskipun bersifat statis dan belum memiliki fitur kompleks seperti database dll, website ini tetap menjadi representasi konkret dari kemampuan dan pencapaian saya selama perjalanan pembelajaran saya.

Desain website ini terinspirasi dari codebayu.com, dengan fokus pada tampilan yang bersih dan intuitif untuk memudahkan pengunjung dalam menjelajahi dan mengenal lebih jauh tentang saya dan project project yang saya buat.
